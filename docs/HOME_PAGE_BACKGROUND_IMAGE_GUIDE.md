## Technical Requirements

### Image Specifications
- **Dimensions:** 1920x1080px (16:9 aspect ratio) minimum
- **Recommended:** 2560x1440px or 3840x2160px for high-DPI displays
- **File Format:** JPG (optimized) or WebP
- **File Size:** Under 500KB (optimized for web)
- **Color Profile:** sRGB

### Image Quality Checklist
- ✅ High resolution (sharp, not pixelated)
- ✅ Good contrast (works with dark overlay)
- ✅ Not too busy (text should be readable)
- ✅ Professional quality
- ✅ Properly lit (not too dark or too bright)
- ✅ Relevant to Ayurvedic clinic

- Best for brand identity

## Image Placement

Once you have the image:

1. **Save it as:** `public/images/hero/home-hero.jpg`
2. **Optimize it:**
   - Use tools like TinyPNG, Squoosh, or ImageOptim
   - Reduce file size while maintaining quality
3. **Test it:**
   - Check how text looks over the image
   - Ensure the dark overlay works well
   - Test on different screen sizes

---

## Current Overlay Settings

The hero section uses:
```css
bg-gradient-to-r from-black/70 to-black/40
```

This means:
- Dark overlay from left (70% opacity) to right (40% opacity)
- Text is white, so the image should work with dark overlay
- You can adjust these values if needed
