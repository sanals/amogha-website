## Platform-Specific Tips

### Leonardo AI
- Use `--style photorealistic` for realistic images
- Use `--style cinematic` for dramatic lighting
- Add `--quality 2` for highest quality
- Example: `[prompt] --ar 16:9 --style photorealistic --quality 2`

### Midjourney
- Use `--v 6` for latest version
- Use `--style raw` for more realistic, less stylized
- Use `--ar 16:9` for widescreen
- Example: `[prompt] --ar 16:9 --style raw --v 6`

### DALL-E 3 (via ChatGPT or Bing)
- Just use the main prompt (no special parameters needed)
- DALL-E automatically handles aspect ratio
- Can specify "16:9 aspect ratio" in the prompt

### Stable Diffusion
- Use negative prompts explicitly
- Set aspect ratio in settings: 1920x1080 or 16:9
- Use realistic checkpoints for best results

## Refinement Tips

### If the image is too dark:
Add to prompt: `bright natural lighting, well lit, soft daylight`

### If the image is too cluttered:
Add to prompt: `minimalist, clean, uncluttered, spacious`

### If you want more traditional feel:
Add to prompt: `traditional Indian architecture, authentic Ayurvedic elements, cultural details`

### If you want more modern feel:
Add to prompt: `modern design, contemporary, sleek, minimalist`

### If colors are too vibrant:
Add to prompt: `muted colors, earth tones, natural palette, soft colors`

---

## After Generation

1. **Download the image** in highest quality
2. **Optimize it:**
   - Resize to 1920x1080px (or 2560x1440px)
   - Compress using TinyPNG or similar
   - Target: Under 500KB
3. **Save as:** `public/images/hero/home-hero.jpg`
4. **Test it** on your website to see how text overlay looks
